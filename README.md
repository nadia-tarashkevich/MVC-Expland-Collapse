# MVC-Expand-Collapse

<!-- Plugin description -->
Quickly navigate your project by expanding the folders you need while automatically collapsing everything else.

**Example:** Click "App" to expand the `app` folder and all its subfolders. Other project branches will collapse to keep the Project View clean. If `app` is already expanded, clicking the button again will collapse only that folder.
<!-- Plugin description end -->

## Features
- **Smart Expansion**: Expand a folder and all its subfolders recursively with one click.
- **Auto-Collapse**: Automatically folds unrelated branches to minimize clutter.
- **Toggle Mechanism**: If the folder is already expanded, clicking the button collapses only that target folder.
- **Dynamic Icons**: Real-time visual feedback on whether a folder is currently expanded or collapsed.
- **Customizable**: Configure up to 5 custom folder names (e.g., "app", "controllers", "views") and bind them to shortcuts in the Keymap.

## Installation
- Using the IDE built-in plugin system:
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> > <kbd>Search for "MVC-Expand-Collapse"</kbd> >
  <kbd>Install</kbd>
